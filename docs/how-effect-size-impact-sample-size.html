15 How Effect Size Impact Sample Size

The following effect size numbers are based on Jacob Cohen’s (Second Edition, p. 82)

15.1.1 Cohen’s

Cohen is used to quantify the size of the difference between two groups, taking into account the variability within each group.

That is, is categorical variable, whereas is numerical variable (i.e., Y is on a continuous scale).

\[d = \frac{m_2 -m_1}{s_{pooled}}=\frac{m_2 -m_1}{\sqrt{\frac{s_1^2+s_2^2}{2}}}\]

15.1.2 Pearson

When both and are numerical, we need to use pearson correlation coefficient to measure effect size .

\[r_{xy} =\frac{cov(x,y)}{s_xs_y}=\frac{\sum_{i=1}^n (x_i-\bar{x})(y_i-\bar{y})}{\sqrt{\sum_{i=1}^n (x_i-\bar{x})^2}\sqrt{\sum_{i=1}^n (y_i-\bar{y})^2}}\]

15.1.3 Convert to

\[d=\frac{2r}{\sqrt{1-r^2}}\]

15.1.4 Convert to

\[r=\frac{d}{\sqrt{4+d^2}}\]

15.2 Effect Size and Sample Size

In general, as the effect size increases, the required sample size decreases, assuming that statistical power (1 - \(\beta\)) and the alpha level (\(\alpha\)) are held constant, respectively.

15.3 R Code Example 1: t-test

To calculate the needed sample size for a two-sample t-test with a specified power of 0.8, a significance level of 0.05, and an effect size of 0.5, you can use the pwr.t.test function in R.

15.4 effect size = 0.50 

# A simple way of writing the R code
library(pwr)
pwr.t.test(n=NULL,
           d = 0.5,
           sig.level = 0.05, 
           power = 0.80, 
           type = "two.sample",
           alternative="two.sided")
## 
##      Two-sample t test power calculation 
## 
##               n = 63.76561
##               d = 0.5
##       sig.level = 0.05
##           power = 0.8
##     alternative = two.sided
## 
## NOTE: n is number in *each* group
