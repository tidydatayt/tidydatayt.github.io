16 Effect Size, Sample Size, and Statistical Power

The following effect size numbers are based on Jacob Cohen’s (Second Edition, p. 82)

16.0.1 Cohen’s

Cohen is used to quantify the size of the difference between two groups, taking into account the variability within each group.

That is, is categorical variable, whereas is numerical variable (i.e., Y is on a continuous scale).

\[d = \frac{m_2 -m_1}{s_{pooled}}=\frac{m_2 -m_1}{\sqrt{\frac{s_1^2+s_2^2}{2}}}\]

16.0.2 Pearson

When both and are numerical, we need to use pearson correlation coefficient to measure effect size .

\[r_{xy} =\frac{cov(x,y)}{s_xs_y}=\frac{\sum_{i=1}^n (x_i-\bar{x})(y_i-\bar{y})}{\sqrt{\sum_{i=1}^n (x_i-\bar{x})^2}\sqrt{\sum_{i=1}^n (y_i-\bar{y})^2}}\]

16.0.3 Convert to

\[d=\frac{2r}{\sqrt{1-r^2}}\]

16.0.4 Convert to

\[r=\frac{d}{\sqrt{d+d^2}}\]

16.1 Effect Size and Sample Size

Sample size is the number of observations in your study.

In general, as the effect size decreases, a larger sample size will be required, assuming that power and alpha are held constant at 0.8 and 0.05, respectively.

16.2 R Code Example 1

To calculate the needed sample size for a two-sample t-test with a specified power of 0.8, a significance level of 0.05, and an effect size of 0.4, you can use the pwr.t.test function in R. Here’s how you can do it:

16.3 effect size = 0.40 

library(pwr)
pwr.t.test(n=NULL,
           d = 0.4,
           sig.level = 0.05, 
           power = 0.80, 
           type = "two.sample",
           alternative="two.sided")
## 
##      Two-sample t test power calculation 
## 
##               n = 99.08032
##               d = 0.4
##       sig.level = 0.05
##           power = 0.8
##     alternative = two.sided
## 
## NOTE: n is number in *each* group

16.5 R Code Example 2

Illustrate the influence of sample size on the p-value in a test.

The fundamental observation is that as the sample size increases, there is a higher probability of obtaining a smaller p-value, assuming a predetermined effect size.

16.6 Sample size = 200 

# Set the seed for reproducibility
set.seed(123)

# Specify parameters
n1 <- 100  # Sample size for group 1
n2 <- 100  # Sample size for group 2
mean_diff <- 4  # Desired mean difference between groups
sd <- 10  # Common standard deviation for both groups

# Generate random samples for two groups
group1 <- rnorm(n1, mean = 0, sd = sd)
group2 <- rnorm(n2, mean = mean_diff, sd = sd)

# Perform a two-sample t-test
t.test(group1, group2)
## 
##  Welch Two Sample t-test
## 
## data:  group1 and group2
## t = -1.5194, df = 197.35, p-value = 0.1303
## alternative hypothesis: true difference in means is not equal to 0
## 95 percent confidence interval:
##  -4.6428618  0.6019159
## sample estimates:
## mean of x mean of y 
## 0.9040591 2.9245320

16.8 Effect Size, Sample Size, and Statistical Power

\(\alpha\) = probability of Type I error, known as a “false positive”

\(\beta\) = probability of Type II error, known as a “false negative”

\(1-\beta\) is also called power, or statistical power.

The basic idea, as sample size increases, the power also increases.

pwr.t.test(n=100,
           d = 0.3,
           sig.level = 0.05, 
           power = NULL, 
           type = "two.sample",
           alternative="two.sided")
## 
##      Two-sample t test power calculation 
## 
##               n = 100
##               d = 0.3
##       sig.level = 0.05
##           power = 0.5600593
##     alternative = two.sided
## 
## NOTE: n is number in *each* group
pwr.t.test(n=200,
